# Getting started

## Installation

Download the `ds` binary from the GitHub releases. Unpack it and move it to your binary folder.

<Tabs>
    <TabItem value="mac" label="Mac" default>
        ```bash
        curl https://github.com/vaubarth/ds2025/releases/download/v1/ds2025-darwin-amd64-v1 -o ds

        chmod +x ds
        ```
    </TabItem>
    <TabItem value="linux" label="Linux">
        ```bash
        curl https://github.com/vaubarth/ds2025/releases/download/v1/ds2025-linux-amd64-v1 -o ds

        chmod +x ds
        ```
    </TabItem>
    <TabItem value="windows" label="Windows">
        ```bash
        iwr -outf ds https://github.com/vaubarth/ds2025/releases/download/v1/ds2025-windows-amd64-v1
        ```
    </TabItem>


</Tabs>

## Configuration

When running `ds` for the first time it will prompt you for some input and create a configuration profile.

```bash
ds

> Whats your name?
> Whats your GitHub username?
> Thats it, now run 'ds auth' to authenticate with dsenv
```

After that you will need to run `ds auth`. This will open a browser window where you need to login with your DS account.
